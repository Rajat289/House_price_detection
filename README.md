# 🏠 House Price Prediction Using Machine Learning

This project is a **Machine Learning-based House Price Prediction system** built using real-world housing data from Bengaluru.

---

## 📂 Files in This Repository

* `Bengaluru_House_Data.csv` → Dataset used for training
* `House_price_detection.ipynb` → Jupyter Notebook with full implementation

---

## 📌 Overview

* Performs **data cleaning & preprocessing**
* Applies **Exploratory Data Analysis (EDA)**
* Builds a **Machine Learning model** to predict house prices
* Demonstrates a complete ML workflow

---

## ⚙️ Tech Stack

* Python 🐍
* Pandas
* NumPy
* Matplotlib / Seaborn
* Scikit-learn

---

## 🚀 Features

* 📊 Data cleaning (handling missing values, outliers)
* 📈 Data visualization (price trends, distributions)
* 🧠 Model training (Regression)
* 📉 Price prediction based on input features

---

## 🧠 Workflow

1. Load dataset
2. Clean and preprocess data
3. Perform EDA
4. Train ML model
5. Evaluate performance
6. Predict house prices

---

## ▶️ How to Run

1. Clone the repository

```bash
git clone https://github.com/Rajat289/house_price_prediction.git
```

2. Open Jupyter Notebook

```bash
jupyter notebook
```

3. Run `House_price_detection.ipynb`
